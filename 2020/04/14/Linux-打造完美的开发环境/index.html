Manjaro 打造舒适的桌面开发系统

Linux

摘要

萝卜青菜各有所爱，Linux 桌面可以说常年是一个大乱斗的环境，谁都不服谁，也没有人能站出来统一天下，也不可能统一，毕竟开源自由是 Linux 的宗旨！这也算 Linux 的魅力之一吧！！！不同于 Windows 和 OS X 的软件生态，Linux 确实弱了那么[一点]，我觉得这么[一点]的范围应该是我们国人日常用些的一些生活娱乐软件，不过在我日常的开发和生活娱乐下，Linux 的软件完全够用，当然如果玩游戏的话那么 Windows 是不二的选择。

Manjaro Linux 是基于 Arch Linux 的 Linux 发行版，使用 Xfce 、KDE Plasma 、GNOME 作为默认桌面环境，当然除了这三个以外还有众多的社区版本提供选择。Manjaro 和 Arch 一样，都采用滚动更新模式。其目标是为 PC 提供易于使用的自由的操作系统。

为什么会选择 Manjaro 这个发行版本呢？

  1. 滚动升级
  2. 自动硬件检测安装，最新的款电脑显卡可能无法兼容（同事最新款联想小新无法兼容）
  3. 先进的包管理、最多软件支持的 Linux 发行版本
  4. 超级高度用户自定义、桌面、主题、界面、文件、操作、显示等等，全部放权给用户自己操作
  5. 软件包版本同步快，基本发行软件版本持平
  6. 超级稳定，兼容性好
  7. Linux 桌面系统排行榜：https://distrowatch.com/

切换源

自动选择源，Manjaro 官方源

# 自动选择源
sudo pacman-mirrors -i -c China -m rank 

# 强制更新源
sudo pacman -Syy

手动配置软件源，非官方源 Arch Linux 国内源，中科大源

# 修改配置文件，添加配置到末尾
sudo vim /etc/pacman.conf

[archlinuxcn]
SigLevel = Optional TrustedOnly
Server = https://mirrors.ustc.edu.cn/archlinuxcn/$arch

# 导入 GPG Key
sudo pacman -Syy && sudo pacman -S archlinuxcn-keyring

# 强制更新包数据，同时更新软件
sudo pacman -Syyu

常用包管理命令

命令 说明
pacman -Syy 强制更新源
pacman -Syyu 强制更新包数据，同时更新软件
pacman -S package 安装包
pacman -Ss package 查询包
pacman -Qs package 查询已安装包
pacman -U package 安装本地包
pacman -U package_url 安装远程包
pacman -R package 删除包，不会删除依赖
pacman -Rs package 删除包，并删除依赖
pacman -Rsc package 删除包，并删除依赖这个包的所有程序（慎用）
pacman -Sc 清理未安装包
pacman -Scc 清理包缓存
yaourt -S package 安装 AUR 包
yaourt -Ss package 查看 AUR 包

Manjaro 包的管理提供了强大的 GUI，命令的用途其实并不大，但是也要了解一下。

初始化

初始化系统，其实就是安装驱动和语言包了，都是傻瓜式的一键安装，但是前提条件是必须系统为最新的状态，如有未更新需要更新后再操作。找到 添加/删除软件查看是否有更新。

找到系统设置->硬件设定->语言包。注：就算不安装驱动和语言包，也不会影响正常使用。完成后注销一下！

Plasma

添加部件推荐：部件也可以在线安装，跟上面操作基本一致

  • application title 可以展示软件图片与软件名称，如没有请安装
  • 全局菜单 跟 OS X 的全局菜单一样而且原生，但有些软件不支持
  • 数字时钟 日期时间展示
  • 任务调度 虚拟工作桌面，不同桌面用不同的软件，视觉效果好
  • 系统托盘 属于一个必备的部件
  • 应用程序面板 可以查看所有程序和一些快捷的操作，非常方便

全局菜单还需要安装一个扩展

sudo pacman -S appmenu-gtk-module
